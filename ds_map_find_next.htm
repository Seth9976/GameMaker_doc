ds_map_find_next

This function returns the next key stored in the DS map after the one specified in the function.

This can be useful if your have to iterate through the DS map looking for something, but should be avoided if possible as it can be slow. If no such key exists then the function will return undefined. You should always check this using the is_undefined function.

 

Syntax:

ds_map_find_next(id, key)

Argument Type Description
id DS Map The handle of the map to use
key String The key to find the next one to

 

Returns:

Any or undefined

 

Example:

var _size = ds_map_size(inventory);
var _key = ds_map_find_first(inventory);
for (var i = 0; i < _size; i++)
{
    if (_key == "gold")
    {
        break;
    }
    
    _key = ds_map_find_next(inventory, _key);
}

The above code creates some temporary variables and then gets the DS map size and finds the first key as stored by the computer in the map. It then uses a for loop to iterate through the DS map looking for the key value "gold". If it finds it, it breaks out of the loop.
