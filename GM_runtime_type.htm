GM_runtime_type

This built-in constant holds the type of the runtime: "gms2" for the current runtime or "gmrt" for the new runtime.

NOTE See GMRT (GameMaker Runtime) for more information about the new runtime.

 

Syntax:

GM_runtime_type

 

Holds:

String

 

Example:

show_debug_message($"Using runtime: {string_upper(GM_runtime_type)}");

The above code outputs a debug message that shows the runtime the game is using.
