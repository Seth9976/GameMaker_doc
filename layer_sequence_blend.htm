layer_sequence_blend

This function sets the blend colour of the given sequence element.

 

Syntax:

layer_sequence_blend(sequence_element_id, blend)

Argument Type Description
sequence_element_id Sequence Element ID The ID of the sequence element
blend Colour The blend colour to use for the sequence element

 

Returns:

N/A

 

Example:

var _arr_elements = layer_get_all_elements("Assets");
for(var i = 0;i < array_length(_arr_elements);i++)
{
    var _element = _arr_elements[i];
    if (layer_get_element_type(_element) != layerelementtype_sequence) { continue; }
    
    layer_sequence_blend(_arr_elements[i], c_red);
}

This code sets the blend colour of all sequence elements on an existing layer named "Assets" to red.
