General Asset Editors

This section covers tools used for asset editing within the IDE, which are not limited to any single type of The Asset Editors.

This includes the following:

  • Code Editor 2 (Beta)
  • The Inspector

Both the Code Editor 2 and the Inspector can be used together at the same time, using the Inspector to edit basic asset information, and the Code Editor 2 to edit text documents in assets such as Objects, Scripts, Notes, etc.

NOTE The Code Editor 2 is currently opt-in. Enable it in the Code Editor 2 Preferences.

While it is disabled, assets with text documents like Objects, Scripts, Notes etc. will open their respective editors, which make use of the Legacy Code Editor.
