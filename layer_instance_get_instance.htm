layer_instance_get_instance

This function can be used to get the unique instance ID of the given instance element. You give the instance element ID (see the code example below for how to get this), and the function will return a handle holding the unique instance id for the element. If the element is not an instance, the function will return an invalid handle (noone).

 

Syntax:

layer_instance_get_instance(element_id)

Argument Type Description
element_id Instance Element ID The unique ID value of the instance element to get the ID of

 

Returns:

Object Instance

 

Example:

var _elements = layer_get_all_elements("Instances");
for (var i = 0; i < array_length(_elements); i++)
{
     if (layer_get_element_type(_elements[i]) == layerelementtype_instance)
     {

         _inst.x = _inst.x + 10;
     }
}

The above code will check get all the instance elements on a layer, then get their unique handles and use that to move them 10px to the right.
