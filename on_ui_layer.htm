on_ui_layer

This built-in variable is created for every instance in a room and stores whether the instance is placed on a UI layer (true) or not (false). You can check the specific type of UI layer by calling collision_space.

 

Syntax:

on_ui_layer

 

Returns:

Boolean

 

Example:

with (all)
{
    if (!on_ui_layer) depth = -bbox_bottom;
}

This runs a line of code for all instances in the room. If the instance is not on a UI layer, its depth is set to its bottom bounding box edge.
