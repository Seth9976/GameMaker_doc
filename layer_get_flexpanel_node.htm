layer_get_flexpanel_node

This function takes the name of a layer as a string. If the layer is a UI layer, it will return the Flex Panel Node for the root of the UI layer, otherwise it will return undefined.

You can use the Flex Panel Functions with the node that is returned.

TIP Use flexpanel_node_get_struct to return the struct for a node. The contents of the returned struct are defined here: Flex Panel Struct Members

 

Syntax:

layer_get_flexpanel_node(layer_name)

Argument Type Description
layer_name String The name of the layer to get the Flex Panel Node for

 

Returns:

Flex Panel Node or undefined

 

Example:

var _ui_layer = layer_get_flexpanel_node("UILayer_1");

var _hearts_flex = flexpanel_node_get_child(_ui_layer, "Hearts");

flexpanel_node_style_set_width(_hearts_flex, 200, flexpanel_unit.point);

This gets the Flex Panel Node of a UI layer and retrieves a child panel called "Hearts". It sets the width of that panel to 200px.
