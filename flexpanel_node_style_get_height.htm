flexpanel_node_style_get_height

This function returns the height property of the given Flex Panel Node.

See: Width and Height / flexpanel_node_style_set_height

 

Syntax:

flexpanel_node_style_get_height(node)

Argument Type Description
node Flex Panel Node The node to read.

 

Returns:

Flex Panel Unit-Value Struct

 

Example:

var _height = flexpanel_node_style_get_height(_node);

This gets the height of a node and stores it in a local variable.
