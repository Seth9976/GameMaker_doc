Particle System Elements

The GameMaker Room Editor allows you to add particle system assets that you create using The Particle System Editor to asset layers. A Particle System Instance is created at runtime for every particle system that you add to a layer in this way. You can also create particle systems on a layer in code using part_system_create_layer. In both cases the newly created particle system is added as an element to the layer. Note that a particle system element corresponds to a Particle System Instance.

To get or set any property of a particle system element on a layer, you first use layer_particle_get_id to get the element ID and then use it in the other functions on this page to get or set the property you want:

var _element_id = layer_particle_get_id("Particles", "particle_1");

layer_particle_x(_element_id, 100);
layer_particle_blend(_element_id, c_red);
var _y = layer_particle_get_y(_element_id);
var _alpha = layer_particle_get_alpha(_element_id);
// Etc.
