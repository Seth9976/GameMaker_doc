The File Watcher

The File Watcher is the part of the IDE that watches the files related to a project for any changes due to external software (i.e.: editing an image in an external editor) or due to internal changes like editing a script asset. It is enabled by default and can be configured in the File Watcher Preferences.

Project Directory Modified Dialog

If the File Watcher detects any changes in your game's project directory, the Project Directory Modified dialog is brought up:

This dialog window lists the detected changes and allows you to choose whether to Reload or Save:

  • Reload: This overwrites any changes there are in memory with what is on disk.
  • Save: This takes what is currently in memory and overwrites what's on disk.

NOTE This window will not be shown if you've enabled Automatically reload changed files in the General Preferences.
