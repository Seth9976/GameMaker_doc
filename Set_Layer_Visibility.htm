 Set Layer Visibility

This action lets you change the visibility of a layer.

The Layer parameter can be the name of the layer as a string or a Layer ID returned from a function like layer_get_id. You then set the visibility of the layer using the checkbox.

 

Action Syntax:

Arguments:

 

Argument Description
Layer The layer to modify
Visible The layer's visibility

 

Example:

The above action will make the "PauseMenu" layer invisible.
