Proxy Settings Preferences

The Proxy Settings Preferences contain the settings related to the proxy.

Some of the fields shown above are optional and if there is one you don't require then you can simply leave the field blank. It is worth noting that if you have set a proxy in your Windows Internet Options, and it doesn't require a username/password, then it should just work "as is" and you probably will not be required to set up the proxy in GameMaker.

If your proxy does require a username and/or password, then you'll need to add them into the fields shown in the window above. Note that in these cases the username/password/proxy details will always be required, and that the proxy IP should also have the port added, i.e.: xx.xx.xx.xx:port.
