 Get Draw Font

With this action you can get the font currently assigned for drawing. The value returned will be either a valid font handle (index >= 0) that represents a font constant from The Asset Browser or an invalid handle (index set to -1) for no font assigned. The return value will be stored in the target variable that you specify, which can have been created previously or can be a new temporary one (if you check the "Temp" check-box).
