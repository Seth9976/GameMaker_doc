mp_grid_get_cell

This function returns whether the given MP grid cell is flagged as occupied or not. If it has been occupied or the position being checked is out of the grid's bounds, the function will return -1, otherwise it will return 0.

 

Syntax:

mp_grid_get_cell(id, x , y)

Argument Type Description
id MP Grid ID Index of the mp_grid that is to be used
x1 Real The x coordinate of the grid to check
y1 Real The y coordinate of the grid to check

 

Returns:

Real

 

Example:

if (mp_grid_get_cell(grid, mouse_x div 16, mouse_y div 16) == -1)
{
    image_blend = c_red;
}
else
{
    image_blend = c_lime;
}

The above code will check the mp_grid cell that corresponds to the mouse position and, if it is occupied, sets the image_blend variable to red, and if it is not occupied it sets it to green.
