image_yscale

This instance variable sets the vertical scaling (along the y-axis) applied to the sprite that has been assigned to the current instance. A scale of 1 indicates no scaling (1:1), smaller values will scale down (0.5, for example, will halve the height of the sprite), larger values will scale up and negative values will mirror the sprite and scale it, unless the value used is exactly -1 (in which case the sprite is just mirrored along the y-axis with no scaling).

NOTE Changing this variable will scale the instance's collision mask and bounding box accordingly. See Collision Mask Rotation & Scaling.

 

Syntax:

image_yscale

 

Returns:

Real (single precision floating point value)

 

Example:

if (image_xscale < 5)
{
    image_xscale += 0.2;
    image_yscale = image_xscale;
}
else
{
    instance_create_layer(x, y, "Effects", obj_Explosion);
    instance_destroy();
}

The above code scales the sprite and then once it is scaled to 5 times its original size, a new instance of another object is created and the instance destroyed.
