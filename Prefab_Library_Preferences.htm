Prefab Library Preferences

The Prefab Library Preferences are used to define how the Prefab Library is displayed (see Prefab Library Features). The following options are available:

  • Layout type: This defines the default layout of the Prefab Library, which can be Horizontal (the default), Vertical, Simple or Tree only.
  • List view type: This option defines how to display the contents of the currently open folder, in a Grid view (the default) or in a List view.
