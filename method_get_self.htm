method_get_self

This function retrieves the instance or struct reference which is the self context used when the method is called. If the variable is not a method then the function will return undefined.

 

Syntax:

method_get_self(method)

Argument Type Description
method Method The method variable to check.

 

Returns:

Object Instance, Struct, or undefined

 

Example:

var _self = method_get_self(light_properties);
show_debug_message(string(_self));

The above code gets the self context for the given method variable and outputs it to the console.
