Animation Curves

Animation curves can be set up in The Asset Browser as assets using The Animation Curve Editor and they can then be used in The Sequence Editor to control tracks in a sequence. However, they can also be used in your code to change values over time in your games for anything from tweening effects on buttons, to movement, to colour changing and more. You can also create your own animation curves at run time using the functions listed here, making them a very flexible tool for your games.
