gpu_get_stencil_ref

This function retrieves the current stencil test reference value set by gpu_set_stencil_ref.

The stencil ref value is used in the stencil test as follows: 

ref cmp_func stencil

NOTE The comparison function used for the stencil test is returned by gpu_get_stencil_func.

 

Syntax:

gpu_get_stencil_ref()

 

Returns:

Real

 

Example:

var _ref = gpu_get_stencil_ref();

The code above gets the current reference value used for the stencil test and stores it in a temporary variable _ref.
