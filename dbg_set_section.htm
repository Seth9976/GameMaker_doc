dbg_set_section

This function sets the current section to a section that was previously created using dbg_section. Any new debug controls will be created in the section set using this function.

 

Syntax:

dbg_set_section(section)

Argument Type Description
section Debug Section Pointer A pointer to a debug section returned by dbg_section

 

Returns:

N/A

 

Example:

var _view1 = dbg_view("View", true);
var _sec1 = dbg_section("Section 1");
var _sec2 = dbg_section("Section 2");
dbg_slider(ref_create(global, "value2"), 0, 20, "Value2");
dbg_set_section(_sec1);
dbg_slider(ref_create(global, "value"), 0, 20, "Value1");

This creates two sections and adds a slider into Section 2. Then it uses dbg_set_section() to set the current section to Section 1 and adds a slider into it.
