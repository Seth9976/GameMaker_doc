flexpanel_node_style_get_align_items

This function returns the alignItems property of the given Flex Panel Node.

See: Align Items / flexpanel_node_style_set_align_items

 

Syntax:

flexpanel_node_style_get_align_items(node)

Argument Type Description
node Flex Panel Node The node to read.

 

Returns:

Flex Panel Alignment Constant

 

Example:

var _align_items = flexpanel_node_style_get_align_items(_node);

This gets the align items value of a node and stores it in a local variable.
